tmux-config
===========

My tmux configuration file

Installation:

1. Clone repo:
```
$ git clone https://github.com/therivenman/tmux-config.git ~/.tmux/tmux-config
```
2. Install `tpm`:
```
$  git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
3. Install `.tmux.conf`:
```
$ ln -s ~/.tmux/tmux-config/tmux.conf ~/.tmux.conf
```
